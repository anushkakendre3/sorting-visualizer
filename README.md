# Sorting Visualizer

An interactive web application for visualizing sorting algorithms through animated bar representations.

## Features

* Bubble Sort
* Selection Sort
* Insertion Sort
* Quick Sort
* Merge Sort
* Adjustable Array Size
* Adjustable Animation Speed
* Real-Time Complexity Information

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Open `index.html` in a browser.
2. Generate a new array.
3. Select a sorting algorithm.
4. Watch the visualization.

## Algorithms Implemented

* Bubble Sort
* Selection Sort
* Insertion Sort
* Quick Sort
* Merge Sort
